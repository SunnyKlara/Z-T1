# 🎨 UI 验收检查清单

> **优先级**: CRITICAL — 每个 UI 组件完成后必须对照此清单检查
> **用途**: 解决"AI 做出来的 UI 不满意"的问题。提供明确的验收标准。
> **创建**: 2026-05-08

---

## 一、视觉检查

### 对齐

- [ ] 所有元素左对齐或居中对齐（不要混用）
- [ ] 同级元素顶部对齐
- [ ] 图标与文字垂直居中
- [ ] 列表项左边缘对齐

### 间距

- [ ] 使用 `ui-design-tokens.md` 中的间距常量
- [ ] 同级元素间距一致
- [ ] 父子元素间距合理（子元素间距 < 父子间距）
- [ ] 页面边距 ≥ 16px

### 颜色

- [ ] 使用 `ui-design-tokens.md` 中的颜色常量
- [ ] 背景与文字对比度 ≥ 4.5:1
- [ ] 状态色使用正确（成功=绿，错误=红，警告=橙）
- [ ] 没有硬编码颜色值

### 字体

- [ ] 使用 `ui-design-tokens.md` 中的字体预设
- [ ] 标题字重 ≥ 600
- [ ] 正文字重 400
- [ ] 没有混用字号（同级文字字号一致）

### 圆角

- [ ] 使用 `ui-design-tokens.md` 中的圆角常量
- [ ] 同级组件圆角一致
- [ ] 按钮圆角 8px
- [ ] 卡片圆角 12px

---

## 二、交互检查

### 按钮

- [ ] 有按下态（颜色变深或阴影变化）
- [ ] 有禁用态（半透明 + 不可点击）
- [ ] 有加载态（显示 loading 指示器）
- [ ] 触摸区域 ≥ 48×48

### 列表

- [ ] 有点击反馈（高亮或水波纹）
- [ ] 有分隔线或间距区分项
- [ ] 右侧有箭头指示可点击
- [ ] 长列表使用懒加载

### 表单

- [ ] 输入框有占位符提示
- [ ] 输入框有聚焦态（边框高亮）
- [ ] 有错误提示（红色文字）
- [ ] 有字数限制提示

### 弹窗

- [ ] 有遮罩层（半透明黑色）
- [ ] 点击遮罩可关闭（除非是确认弹窗）
- [ ] 有关闭按钮
- [ ] 内容不超过屏幕 80%

---

## 三、状态检查

### 加载状态

- [ ] 页面加载有全屏 loading
- [ ] 局部加载有局部 loading
- [ ] 按钮加载有按钮内 loading
- [ ] loading 居中显示

### 空状态

- [ ] 无数据时有图标 + 文案
- [ ] 有引导操作按钮
- [ ] 文案友好（不是"无数据"）

### 错误状态

- [ ] 有错误图标
- [ ] 有错误描述
- [ ] 有重试按钮
- [ ] 文案使用用户语言

### 成功状态

- [ ] 有成功图标
- [ ] 有成功提示
- [ ] 自动消失或可手动关闭

---

## 四、动画检查

### 页面切换

- [ ] 有过渡动画（≤300ms）
- [ ] 动画流畅无卡顿
- [ ] 切换方向正确（前进/后退）

### 组件动画

- [ ] 出现/消失有淡入淡出（≤200ms）
- [ ] 列表项有交错动画
- [ ] 按钮有按下反馈（≤100ms）

### 加载动画

- [ ] 使用 CircularProgressIndicator 或自定义动画
- [ ] 动画流畅
- [ ] 不会无限旋转（操作完成后停止）

---

## 五、响应式检查

### 屏幕适配

- [ ] 不同屏幕尺寸显示正常
- [ ] 横屏/竖屏都可用
- [ ] 文字不会溢出屏幕
- [ ] 图片不会变形

### 字体缩放

- [ ] 系统字体放大时布局正常
- [ ] 文字不会重叠
- [ ] 按钮文字完整显示

---

## 六、可访问性检查

- [ ] 所有图标有语义标签
- [ ] 颜色不是唯一的信息传递方式
- [ ] 触摸区域 ≥ 48×48
- [ ] 文字对比度达标

---

## 七、UI 验收流程

```
1. AI 完成 UI 组件
   ↓
2. 对照此清单逐项检查
   ↓
3. 标记不通过项
   ↓
4. 修复不通过项
   ↓
5. 重新检查直到全部通过
   ↓
6. 输出验收报告
```

### 验收报告模板

```
═══════════════════════════════════════════════════════════════
🎨 UI 验收报告

组件: [组件名称]

视觉检查:
  [x] 对齐正确
  [x] 间距合理
  [x] 颜色使用令牌
  [x] 字体使用预设
  [x] 圆角一致

交互检查:
  [x] 按钮有按下态
  [x] 列表有点击反馈
  [ ] ❌ 问题: 弹窗点击遮罩不能关闭

状态检查:
  [x] 加载状态正确
  [x] 空状态友好
  [x] 错误提示清晰

动画检查:
  [x] 页面切换流畅
  [x] 组件动画自然

决策: [需要修复 / 通过]
═══════════════════════════════════════════════════════════════
```

---

*每个 UI 组件必须通过此清单才能标记为完成*
